## 📌 Overview

A personal portfolio website built to showcase my skills, projects, and background as a Front-End Developer. Features a stunning **galaxy video background** and a clean, modern layout designed to make a lasting impression.

---

## ✨ Features

- 🎥 **Galaxy Video Background** — immersive and eye-catching hero section
- 📱 **Fully Responsive** — works seamlessly on desktop, tablet, and mobile
- 🎨 **Modern UI Design** — clean layout with smooth visual flow
- 🧭 **Smooth Navigation** — easy-to-use section-based layout
- ⚡ **Pure HTML & CSS** — no frameworks, no dependencies, lightning fast

---

## 🛠️ Built With

| Technology | Purpose |
|-----------|---------|
| HTML5 | Page structure & semantic markup |
| CSS3 | Styling, layout & animations |
| Video (MP4) | Galaxy background effect |
